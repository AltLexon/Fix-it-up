# Fix It Up Script

This repository contains a script made for the game **Fix It Up** in Roblox.  
It includes systems for car management, teleportation, player modifications, auto-buying, shop utilities, ESP, and UI configuration.

---

## **Car**
- Current Car: Car Name  
- Current Car Condition  
- Can Sell: ✅ / ❌ (with timer)  
- Spawn Own Car  
- Spawn Current Car  
- Teleport to Car  
- Car Speed Multiplier  
- Sell Car Button  
- Repair Car Button  
- Random Color on Repair (checkbox)  
- Paint Car Custom Color  
- Enable Parts ESP  

---

## **Junkyard**
- Enable Cars ESP  
- All junkyard vehicles listed with tiers: **S, A, B, C, D**

---

## **Auto**
- Auto Buy  
- Auto Repair On Buy  
- Select Car To Auto Buy  

---

## **Shop**
- Buy anything from the shop store  

---

## **Player**
- Change player speed  
- Reset player speed  

---

## **Teleport**
- Teleport to Garage  
- Teleport to Waypoints  
- Teleport to other players  

---

## **Config**
- Change UI Theme  
- Hitbox Config  
  - Fill Color  
  - Outline Color  
  - Text Color  
- Rejoin
